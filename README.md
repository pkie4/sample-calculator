Simple Python Calculator

Files:
- calculator.py : Lightweight safe evaluator and REPL.

Usage:

Interactive REPL:

```bash
python calculator.py
```

One-shot evaluation:

```bash
python calculator.py "2 + 3*4"
```

Notes:
- Only basic arithmetic, unary operators, and a handful of math functions are allowed for safety.
- Type `exit` or `quit` to leave the REPL.
